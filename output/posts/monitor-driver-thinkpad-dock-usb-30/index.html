Last week I received the last piece to get up and running my ThinkPad USB-3.0 docking station. While, the configuration worked out of the box for windows systems, configuring the docking station with Ubuntu OS was not straightforward.

It turned out that the Ubuntu OS needed an installation of an external driver in order to properly work with the docking station.

As always stack-overflow was a useful friend and I was manage to find this useful link outlining the necessary steps in order to download the necessary dependencies and install the needed driver.

Once that is done your docking station should work out of the box.

You can then look at the connected monitors using the terminal command 

xrandr -q

And once you find the monitor, which you might want to change the orientation you can run 

## xrandr --output <monitor name>  --rotate <direction>

xrandr --output DVI-I-2-2 --rotate left
